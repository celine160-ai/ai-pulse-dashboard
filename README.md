## Features

- **Macro Trend Cards** – Displays key insights on agentic interoperability, MCP infrastructure, and action-first UI.
- **Claude Skills Ecosystem** – Curated list of trending Claude-related skills with direct links.
- **GitHub Open Source (7d)** – Fetches top AI repositories from the last 7 days, sorted by stars.
- **Product Hunt Daily** – Shows the latest products from Product Hunt with descriptions.
